
## Nincs CS:GO config.
Based on budi's CS:GO config

1. Extract autoexec.cfg & buybinds.cfg to \SteamLibrary\steamapps\common\Counter-Strike Global Offensive\csgo\cfg
2. Start the game
3. Look in the developer console for "autoexec.cfg executed".
